## 2.1.7
选择排序更快。对于长度为N的数组。
* 选择排序需要N/2次交换，(1+N-1)*(N-1)/2次比较。
* 插入排序需要(1+N-1)*(N-1)/2次交换，(1+N-1)*(N-1)/2次比较。
